### 文本格式化
>b和strong的区别
- 纯粹物理上的字体加粗作用（不优先使用）
- 字体加粗作用，但具有强调的意义，属于语义化标签,可以被浏览器等更好检索

>i和em的区别
- 显示效果一样
- i标签纯粹斜体显示，em标签强调的作用，属于语义化标签

## 缩写标签
abbr 悬浮在缩写上面，会展示全称

## 容器标签
div
> div是个块级标签，主要作用用来组合其他html元素，提供一个容器。无语义化，用来做元素上的分组

## 列表标签

> 无序列表
>
> -  type值square、disc、circle 指定列表着重号样式


>有序列表
| 属性名 | 属性值 | 说明             |
| ------ | :----- | ---------------- |
| type   | "1"    | 阿拉伯数字排序   |
|        | "a"    | 小写字母排序     |
|        | "A"    | 大写字母排序     |
|        | "i"    | 小写罗马数字排序 |
|        | "I"    | 大写罗马数字排序 |

|       |              |                        |
| ----- | ------------ | ---------------------- |
| start | 值是一个整数 | 指定有序列表的起始编号 |

> 自定义列表
- dl： Definition List（定义列表）
- dt： Definition Term（定义术语）
- dd： Definition Description（定义描述）
